# Modern Blog Platform

## Overview
This is a modern blog platform homepage designed with a creative grid layout and sophisticated serif typography. It features a responsive posts grid, a search bar with instant filtering, and a newsletter subscription widget.

## Features
- Creative grid layout for blog posts
- Sophisticated serif typography using Google Fonts
- Featured article cover with read-time and author badge
- Instant search filtering for posts
- Categories tab menu
- Responsive design for all devices
- Newsletter subscription widget

## Technologies Used
- HTML
- CSS
- JavaScript
- Google Fonts

## Setup Instructions
1. Clone the repository or download the files.
2. Open `index.html` in your web browser to view the blog platform.
3. Use the search bar to filter posts by title.
4. Subscribe to the newsletter by entering your email.
